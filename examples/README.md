# Examples Guidelines

This directory contains sanitized, public-facing examples that demonstrate workflow behavior and evaluation structure.

## NSFW / Adult-Content Testing

This repository uses sanitized examples for public documentation.

For adult visual novel workflows, examples should preserve the narrative function of intimacy, desire, tension, jealousy, vulnerability, or erotic atmosphere without including explicit sexual prose.

Do not commit private NSFW fixtures, copyrighted text, or material involving minors, coercion, or ambiguous consent.

Keep explicit model-evaluation fixtures local and gitignored.

## Private Fixture Format (Local Only)

Do not commit explicit fixtures to this repository.

- TARGET EXPERIENCE
- SCENE CONTEXT

- CONSENT / BOUNDARIES

- EVALUATION QUESTION
